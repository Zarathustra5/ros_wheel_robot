# ros_wheel_robot

## Запуск проекта
Убедитесь, что находитесь в корне проекта (mikita@ubuntu:~/catkin_ws) и запустите скрипт старта проекта:

```
sh start.sh
```
Откроется несколько терминалов, которые запустят все необходимое и появится 2 окна - rqt для управления роботом и rviz для отображения визуальной симуляции.

Для управления роботом в rqt нужно открыть Plugins ->Robot Tools -> Robot Steering, также поменять /cmd_vel на /controller/cmd_vel.

После внесения изменений в код нужно закрыть все окна и перезапустить скрипт старта проекта.
